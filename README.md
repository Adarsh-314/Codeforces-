## 🚀 Getting Started

### Prerequisites
- C++ compiler (GCC, Clang, or MSVC)
- Basic understanding of DSA concepts

### Compiling and Running

To compile a solution:
```bash
g++ -o cdf1 cdf1.cpp
```

To run a compiled solution:
```bash
./cdf1
```

Or compile and run in one command:
```bash
g++ -o cdf1 cdf1.cpp && ./cdf1
```
